# StyleCore

StyleCore is a modular React + Vite + Tailwind dashboard for AI-driven visual style exploration and prompt construction.

## Install

```bash
npm install
```

## Run (development)

```bash
npm run dev
```

## Build (production)

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```

## Lint

```bash
npm run lint
```
